# Inventory Management System (InventoryApp)

A full-stack Inventory Management System built to manage stock levels, track inventory adjustments, and provide real-time low-stock notifications.

This project was developed as a portfolio project to demonstrate practical, real-world full-stack development skills using the MERN stack with Firebase Authentication and Socket.IO.

---

## Key Features

- User authentication with Firebase
- Role-based access control (User / Admin)
- Inventory creation and management
- Stock adjustments with full ledger history
- Inventory ledger report (PDF export)
- Real-time low stock notifications using Socket.IO
- Protected routes with authentication and authorization
- Clean, responsive UI built with Tailwind CSS

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Firebase Authentication
- Axios
- Socket.IO Client

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- Socket.IO
- JWT-based authorization
